# 📊 Power BI Customer Analysis Project

## 📌 Project Overview
This project analyzes customer shopping behavior using Python and Power BI. The dataset was cleaned and transformed using Python before building an interactive dashboard in Power BI to uncover customer purchasing patterns and business insights.

---

## 🎯 Objectives
- Analyze customer shopping behavior.
- Clean and preprocess raw data.
- Build an interactive Power BI dashboard.
- Identify trends and key business insights.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Jupyter Notebook
- Power BI
- CSV Dataset
